studysheet
==========

StudySheet is a python script that takes a list of words,
and creates a study sheet based on knowledge and definitions
from Wikipedia or WolframAlpha.

## Milestone Tasks
1. Integrate with WikiMedia API
2. Integrate with WolframAlpha API
3. Add script arguments for main
4. Save results as a pdf through PyTeX
5. Create Flask+Angular webapp
6. Cache results
7. Add image support
